# UB Sofia
* Sofia Learns About Research. Educational quiz game, adapted for web release.

### Version
* Haxe 3.4.3
* Flambe 4.1.0_haxe_3.4.3

### Maintainers
* Robert Adams


# Getting Started
1 - To Run
In First Terminal:
> flambe serve

In Second Terminal:
> flambe run html --debug

2 - To make a Build
In Second Terminal:
> flambe build html

3 - Others

To Clean a Build:
In Second Terminal:
> flambe clean

To View Haxe Library Setup
> haxelib list

For those who uses Visual Studio:
<< Important Note >>
Flambe may hang on Build
> To prevent a build from hanging while VSCode is open, go to VSCode > Preferences > Settings, and add "haxe.displayPort": 9000
